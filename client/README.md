# Frontend

## Svelte
Frontend Framework & UI Framework

- https://svelte.dev/
- https://sveltematerialui.com/

## Bun
JavaScript runtime

- https://bun.sh/

## Set Up
```bash
bun install
```

## Developing

```bash
bun run dev
```

## Building

```bash
bun run package
```

To create a production version of your showcase app:

```bash
bun run build
```
